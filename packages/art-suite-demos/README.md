# Art.React Demos

Get a look at what ArtReact and the ArtEngine can do.

### Install and Run

```
git clone https://github.com/Imikimi-LLC/art-react-demos.git
cd art-react-demos
npm install
npm run hot
```

Go to: [http://localhost:8080/webpack-dev-server/](http://localhost:8080/webpack-dev-server/)
